▪ Προσθήκη ενός νέου χρήστη - H εισαγωγή νέου χρήστη/κόμβου στο δυαδικό δέντρο
αναζήτησης θα γίνεται με βάση το User Name του χρήστη. Δεν θα πρέπει να επιτρέπεται
η καταχώριση νέου χρήστη με User Name ίδιο με άλλου χρήστη στο κοινωνικό δίκτυο.

▪ Ενημέρωση συνδέσεων μεταξύ των χρηστών - Σε περίπτωση που ένας χρήστης του
δικτύου κάνει αποδεκτό ένα αίτημα φιλίας από άλλον χρήστη θα πρέπει να ενημερωθούν
οι αντίστοιχες συνδέσεις/επαφές των χρηστών που εμπλέκονται, στους κατάλληλους
κόμβους του δέντρου.
//n ^ 

▪ Αναζήτηση χρήστη και εμφάνιση των στοιχείων του - Η αναζήτηση θα πρέπει να γίνεται με
βάση το User Name του. Αν η αναζήτηση είναι επιτυχής θα πρέπει να εμφανίζονται όλα
τα στοιχεία του χρήστη καθώς και οι επαφές του. Σε περίπτωση που δεν υπάρχει χρήστης
με αυτό το User Name, θα πρέπει να εμφανίζεται κατάλληλο μήνυμα.


▪ Διαγραφή χρήστη - Η διαγραφή θα πρέπει να γίνεται με βάση το User Name του χρήστη.
//n ^

▪ Εμφάνιση όλων των χρηστών του κοινωνικού δικτύου - Η εμφάνιση θα πρέπει να γίνεται
αλφαβητικά σύμφωνα με το User Name των χρηστών.
//n ^

▪ Εμφάνιση του πλήθους των χρηστών του κοινωνικού δικτύου.

▪ Έλεγχος αν δύο χρήστες του κοινωνικού δικτύου είναι φίλοι μεταξύ τους - Ο χρήστης της
εφαρμογής θα δίνει τα User Name δυο χρηστών και η εφαρμογή θα εμφανίζει σχετικό
μήνυμα αν τους συνδέει σχέση φιλίας ή όχι στο τρέχων κοινωνικό δίκτυο.
//n ^

▪ Εύρεση των κοινών επαφών (φίλων) μεταξύ δύο χρηστών του κοινωνικού δικτύου – Ο
χρήστης της εφαρμογής θα ορίζει τα User Name δυο χρηστών και η εφαρμογή θα
εμφανίζει τους κοινούς φίλους που έχουν. Αν δεν υπάρχουν κοινοί φίλοι θα εμφανίζεται
σχετικό μήνυμα.
//n ^
